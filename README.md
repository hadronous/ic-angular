## Contributing

### Setup

- Install [NodeJS](https://nodejs.org/en) v20 (LTS)
- Install [commitizen](https://commitizen-tools.github.io/commitizen/)
- Install [pre-commit](https://pre-commit.com/)
- Activate PNPM:
  ```bash
  corepack enable
  ```
- Install dependencies:
  ```bash
  pnpm i
  ```
