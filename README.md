## Contributing

### Setup

- Install [pnpm](https://pnpm.io/installation)
- Install [NodeJS](https://nodejs.org/en)
- Install [commitizen](https://commitizen-tools.github.io/commitizen/)
- Install [pre-commit](https://pre-commit.com/)
- Install dependencies:
  ```bash
  pnpm i
  ```
